mywebglflightspace a program by NGUYEN.Chung (freeware 2015)



click on mywebglflightspace.html to run program in localhost mode using local files.(nb: chrome doesnt load local images)


arrows => turn ,  ctrl+arrows or pageup/pagedown => motor , H => help  , esc => quit 

P => pause , T => change timehour , V => view , shift+arrows => turn head (cockpit view)
 
w => enter planet/space orbit



radar dots : dark red (lower spaceship) -> red (same azimut) -> yellow (upper spaceship)



uses webaudio and webgl (check if your browser is compatible, or use firefox)
